✂ 키보드 스티커 인쇄 & 사용 가이드

인쇄 방법

  1. 브라우저에서 Cmd + P 로 인쇄 창 열기
  2. 용지 크기: A4 / 방향: 세로(Portrait)
  3. 여백: 없음(None)
  4. 배경 그래픽: 반드시 활성화
  5. 스티커 용지(잉크젯/레이저 호환)에 출력

스티커 사용법

  • 점선(---) 따라 가위로 오려내기
  • 키 옆면/상단에 작은 스티커 부착
  • 큰 스티커는 손목받침대/모니터 베젤에
  • 색상별 카테고리로 분류되어 있음
  • 마우스패드 레이아웃과 함께 사용 추천
